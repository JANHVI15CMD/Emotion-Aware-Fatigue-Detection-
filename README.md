#  Emotion-Aware Fatigue Detection System

An AI-powered real-time fatigue and emotion detection system that monitors facial expressions, eye movement, and emotional states using Computer Vision and Machine Learning techniques. The system helps identify signs of drowsiness, fatigue, and emotional stress through webcam-based live monitoring.

---

##  Features

*  Real-time Emotion Detection
*  Fatigue / Drowsiness Detection
*  Eye State Monitoring
*  Live Webcam Processing
*  Machine Learning Based Prediction
*  Real-Time Alert System
*  Emotion-Aware Monitoring

##  How It Works

The system captures live video through a webcam and analyzes facial features such as:

* Eye movement
* Facial expressions
* Blink patterns
* Emotional states

Using Computer Vision and Deep Learning models, the system predicts:

* Fatigue level
* Drowsiness
* User emotion

If fatigue or drowsiness is detected, the system can trigger alerts in real time.
